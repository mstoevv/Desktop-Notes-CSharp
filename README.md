# Note Taking App 📝

## Overview
[cite_start]Основната цел на проекта е разработването на настолно приложение, което позволява на потребителите да създават, редактират, изтриват и съхраняват бележки[cite: 20]. [cite_start]Приложението предоставя личен достъп чрез система за регистрация и вход[cite: 21].

## Key Features
* [cite_start]**User Authentication:** Регистрация и вход с потребителско име и парола[cite: 29, 30].
* [cite_start]**Personal Space:** Всеки потребител вижда само своите собствени бележки[cite: 43].
* [cite_start]**CRUD Operations:** Пълно управление на бележки (създаване, четене, редакция, изтриване)[cite: 32, 54].
* [cite_start]**Secure Storage:** Данните се съхраняват сигурно в база данни[cite: 67].

## Tech Stack
* [cite_start]**Language:** C# [cite: 61]
* [cite_start]**UI Framework:** Windows Forms (WinForms) [cite: 58]
* [cite_start]**Database:** Microsoft SQL Server Express [cite: 59]
* [cite_start]**IDE:** Visual Studio 2022 

## Architecture & Database
[cite_start]Приложението използва клиент-сървър архитектура[cite: 45]. Базата данни включва:
* [cite_start]`Login_new`: Таблица за потребителски профили[cite: 49].
* [cite_start]`UserNotes`: Таблица за съхранение на бележките, свързани с конкретен потребител[cite: 50].
