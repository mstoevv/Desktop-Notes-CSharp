# Note Taking App 📝

## Overview
Основната цел на проекта е разработването на настолно приложение, което позволява на потребителите да създават, редактират, изтриват и съхраняват бележки. Приложението предоставя личен достъп чрез система за регистрация и вход.

## Key Features
* **User Authentication:** Регистрация и вход с потребителско име и парола.
* **Personal Space:** Всеки потребител вижда само своите собствени бележки.
* **CRUD Operations:** Пълно управление на бележки (създаване, четене, редакция, изтриване).
* **Secure Storage:** Данните се съхраняват сигурно в база данни.

## Tech Stack
* **Language:** C# 
* **UI Framework:** Windows Forms (WinForms)
* **Database:** Microsoft SQL Server Express 
* **IDE:** Visual Studio 2022 

## Architecture & Database
Приложението използва клиент-сървър архитектура. Базата данни включва:
* `Login_new`: Таблица за потребителски профили.
* `UserNotes`: Таблица за съхранение на бележките, свързани с конкретен потребител.
